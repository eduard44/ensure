<static> requireIsNewThis(constructor, context)

Check if a constructor was called with the "new" keyword

Useful during development for preventing constructor functions messing up the global context when they are called without the new keyword

This function will throw an error when your object is constructed without using new

Parameters:
Name Type Description
constructor function

Constructor function to protect
context Object

this context of the object
Source:
  • ensure.js, line 493
Example
// Use inside a constructor
var myClass = function () {
    ensure.isNewThis(myClass, this);
}

// Works fine
var myInstance = new myClass();

// Throws an exception
var myInstance2 = myClass();

Wrap around a function and perform type checks

Parameters:
Name Type Argument Description

Array of the types of each argument in the function, in the same order

Type that the function should return

Source:
  • shield.js, line 36
Returns:
Type
function
Example
// First we create our shielded function
var myShieldFunction = ensure.shield([Boolean, Array], Number, function (arg1, arg2) {
    if (arg1) {
        return {};
    }

    return 1337;
});

// This works fine
myShieldFunction(false, []);

// This throws an error
myShieldFunction([], []);

// This also throws an error since the return value is not a number
myShieldFunction(true, []);
